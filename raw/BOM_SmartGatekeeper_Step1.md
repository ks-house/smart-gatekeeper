## BOM 검토 메모

### ✅ 호환성 확인 항목

| 항목 | 결론 |
|------|------|
| ESP32-C6 + VL53L0X (3.3V) | ✅ 전압 호환. GPIO6/7 I2C 배선 확정 |
| 릴레이 Low/High Trigger 선택형 | ✅ **Open Q#2 해소**: 모듈 점퍼로 Active-LOW 선택 가능. config.h `RELAY_ACTIVE_LOW=true` 유지 |
| 릴레이 3.3V 신호 입력 | ✅ 광절연(오토커플러) 내장 모듈이므로 GPIO 직결 안전 |
| 5V 2A 전원 | ✅ ESP32-C6(~500mA) + VL53L0X(20mA) + 릴레이 코일(~70mA) 합계 < 600mA, 충분 |
| ABS 박스 100×60×40mm | ✅ ESP32-C6 DevKitC-1 크기(51×25mm) + VL53L0X + 릴레이 수납 가능 |

### ⚠️ 주의 사항 2개

| # | 항목 | 내용 | 권고 조치 |
|---|------|------|---------|
| W1 | 400홀 미니 브레드보드 | ESP32-C6 DevKitC-1은 핀이 15×2=30핀으로 400홀 미니를 **가득 채움** → 주변 부품 연결 공간 부족 가능 | 800홀 풀사이즈 브레드보드 추가 구매 권장. 없으면 ESP32-C6를 브레드보드 한쪽 끝에 배치하고 나머지 공간 활용 |
| W2 | 점퍼 와이어 M-M 미언급 | 브레드보드 간 연결에는 **수-수(Male-to-Male)** 필수. M-F, F-F만으론 브레드보드 내 배선 불가 | 구매한 세트에 M-M 포함 여부 확인. 없으면 추가 구매 |

### 릴레이 점퍼 설정 방법 (현장 확인 필요)

```
대부분의 Low/High Level Trigger 선택형 모듈:
  - 보드 위 점퍼 핀 또는 패드에 "H" / "L" 레이블 있음
  - 점퍼를 "L" (LOW) 위치에 꽂으면 → Active-LOW 모드
    (config.h RELAY_ACTIVE_LOW = true 와 일치)
  - 배선 전 반드시 확인 후 config.h 값 최종 확정
```
